## Hướng dẫn release/deploy site bằng github
1. Truy cập vào repo bạn muốn deploy
2. Chọn vào tab setting
3. Ở menu bên trái chọn pages
4. Trong phần GitHub Pages > Source: chọn nhánh muốn deploy (master/main)
5. Bấm vào nút Save
